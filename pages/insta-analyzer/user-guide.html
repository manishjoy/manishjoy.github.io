❒As we are going to use the method which doesn't impact your Instagram profile. So we need to use the official Instagram process to download your data which is the most secure way.

❒Use this link to download your data: Dowload Link & Instruction 

❒Just make sure to choose JSON format in the last step.

❒Once you are done with the steps, you will get an email to download your instagram data in a compressed zipped file.

❒Download the file in your device (computer, laptop etc.). Extract/De-compress the file.

❒Now, in the de-compressed folder, you will find several other folders. Go to connections ⟶ followers_and_following

❒You will find two files, following.json and followers_1.json (Depending upon the number of following and follow, you may find more than one file like followers_2.json, followers_3.json and so on)

❒This application accepts only one JSON file for followers as well as following. If you see more than one followers or following files, you will have to merge them into one. Use this tool to do So: JSON Merger . Make sure to merge similar kind of JSON data only, e.g. followers_1.json with followers_2.json, followers_3.json

❒Now use this application , upload your followers and following files in the respective fields.

❒Click on the Submit button.

❒The application will analyze the data and will generate a text file with the Instagram IDs whom you follow and they don't follow you back.

❒And, don't worry. This application doesn’t save your data, so your data is completely safe with you.
